# Hospital_Management_System
Project Overview

The Hospital Management System is a full-stack web application designed to streamline and automate hospital operations. The system provides an efficient platform for managing patients, doctors, appointments, prescriptions, billing, and medical records in a centralized database. It reduces paperwork, improves data accuracy, and enhances communication between patients, doctors, and hospital administrators.

The application allows patients to register, book appointments, and access their medical information online. Doctors can manage patient records, prescribe medications, and monitor treatment history. Administrators can oversee hospital activities, manage staff information, generate reports, and maintain overall system operations.
